Landing Page Animada

Sobre o Projeto
Landing page desenvolvida para apresentar informações de forma clara, moderna e responsiva. O projeto inclui animações ao rolar a página, efeitos visuais e um formulário de contato para interação com os usuários.

Funcionalidades

Layout responsivo
Animações ao rolar a página
Efeito no cabeçalho durante a rolagem
Formulário de contato
Seção de apresentação dos desenvolvedores

Tecnologias Utilizadas

HTML
CSS
JavaScript

Nível de Conhecimento Aplicado

HTML: Iniciante
CSS: Iniciante
JavaScript: Iniciante

Habilidades Aplicadas

Estruturação de páginas com HTML semântico
Estilização com CSS moderno
Criação de animações e efeitos visuais
Desenvolvimento de layout responsivo
Introdução à interatividade com JavaScript
